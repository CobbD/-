# 如何安装npm<a id="orgheadline1"></a>

参考 [这个网页](http://blog.npmjs.org/post/85484771375/how-to-install-npm) ，如果你nodejs已经安装了，那么安装npm只需要简单输入:

    sudo npm install npm -g
